# Laporan Pelayanan Masyarakat

A simple web-based reporting system built using **PHP** and **MySQL**.
This project allows users to submit, manage, and track public service
reports in an organized and efficient way.

## Features

-   Submit laporan/pengaduan masyarakat
-   Manajemen data laporan (Create, Read, Update, Delete)
-   Login Admin (Session-based authentication)
-   Dashboard ringkas untuk melihat statistik laporan
-   Pencarian & filter laporan
-   Upload bukti foto (optional)

## Tech Stack

-   PHP 7+ / 8+
-   MySQL / MariaDB
-   HTML5, CSS3, JavaScript
-   Bootstrap (optional)
-   XAMPP / Laragon / Localhost server
